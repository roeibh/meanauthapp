MEAN Stack

We're using mongoDB, nodejs, express and Angular to run our application. All written in TypeScript, using TypeGoose as our way to connect to mongoDB (similar to mongoose, but with TypeScript support)

Safe and Secure

We're using JWT standard method for representing claims securely between the client and the server. All sensetive information is being stored securly in the database using argon2 as the cryptographic function.
